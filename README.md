# Assignment 1 - Circular Buffer in C

تم تنفيذ مخزن دائري بسعة محددة واختباره مع إدخال اسم المستخدم وإلحاق “CE-ESY”.

## العمليات المنفذة:
- تهيئة buffer
- كتابة (write) مع التحقق من الامتلاء
- قراءة (read) مع التحقق من الفراغ
- إظهار الناتج وتفريغ المخزن بالكامل

## كيفية التشغيل:
- غير SIZE لاختبار حالات الامتلاء
- أدخل اسمك عند التشغيل
- سيعرض البرنامج النص المُخزّن بعد قراءته
